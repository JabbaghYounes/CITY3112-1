# High Performance Computing

Collectiong of scripts to setup enviornment, benchmark hardware and visualise results for local hardware testing

## Demo

![Demo GIF](https://via.placeholder.com/600x300?text=Demo+GIF)

## Installation
### Using Git
```bash
git clone https://github.com/JabbaghYounes/CITY3112-1.git
```

## Usage
### 1.
```bash
chmod -R +x /scripts
```
### 2.
```bash
./start.sh
```
### 3.
```bash
./benchmark-gpu.sh
```
### 4.
```bash
./visualise.sh
```
### 5.
```bash
./stop.sh
```
